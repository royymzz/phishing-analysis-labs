# phishing-analysis-labs
This repository contains beginner-level phishing analysis exercises
performed in Kali Linux.

## Skills practiced
- Phishing email analysis
- Domain investigation (whois, dig)
- URL inspection
- Indicator of Compromise (IOC) extraction
- Defensive recommendations

## Tools
- Kali Linux
- whois
- dig
- curl
- whatweb

## Purpose
Learning ethical hacking and phishing detection in a safe and legal way.
